# Sprint Contract

Define acceptance criteria before starting work.

1. Read current context:

!cat docs/TODO.md 2>/dev/null
!cat docs/SPRINT.md 2>/dev/null

2. Write a CONTRACT to docs/SPRINT.md:
   - Feature name
   - Acceptance criteria (testable, specific)
   - Verification steps (exact commands)
   - Files to modify
   - Out of scope
   - Estimate (S/M/L)

3. Do NOT start coding until contract is confirmed.
