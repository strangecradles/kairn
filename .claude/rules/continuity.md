# Continuity

After every significant decision or discovery:

1. Update `docs/DECISIONS.md` with:
   - What was decided
   - Why (alternatives considered)
   - Date

2. Update `docs/LEARNINGS.md` with:
   - Non-obvious behavior discovered
   - Gotchas or footguns
   - Patterns that worked well

3. Update `docs/TODO.md` task status as work progresses.

4. If you correct Claude on a mistake, add it to the **Known Gotchas** section in CLAUDE.md.

These files are the project memory. Keep them current.
