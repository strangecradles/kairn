# Kairn Templates

Gold-standard reference templates for every file type Kairn generates.

These are bundled into the compilation prompt so the LLM has concrete examples of what high-quality output looks like — not just instructions.

## Files

- `claude-md.md` — CLAUDE.md template (7-section structure)
- `commands.md` — slash command templates (with shell integration)
- `rules.md` — rules templates (unconditional + path-scoped)
- `skills.md` — skill templates (TDD, debugging, research)
- `agents.md` — subagent templates (reviewer, tester, researcher)
- `hooks.md` — settings.json hook patterns
- `settings.md` — settings.json permission/security patterns
